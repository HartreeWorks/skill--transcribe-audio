# Transcribe Audio

A Claude Code skill for transcribing audio files using Parakeet MLX (fast, runs on Apple Silicon).

## Documentation

See [SKILL.md](./SKILL.md) for complete documentation and usage instructions.

## Installation

To use this skill with Claude Code, clone it into your skills directory:

```bash
git clone https://github.com/HartreeWorks/skill--transcribe-audio.git ~/.claude/skills/transcribe-audio
```

Then restart Claude Code to load the skill.
